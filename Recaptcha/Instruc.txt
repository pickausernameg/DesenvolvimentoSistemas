1 - O arquivo app.py contém o código responsável por renderizar a API que o google disponibiliza. O arquivo form.html contém um exemplo de como renderiza a API.
2 - Caso não tenha, crie uma conta no google para acessar a API do Recaptcha, e através desse link (https://developers.google.com/recaptcha/intro) siga as instruções para receber a chave secreta.
3 - Depois de concluído o passo 3, coloque seu RECAPTCHA PUBLIC KEY recebido como valor do campo "app.config['RECAPTCHA_PUBLIC_KEY']", no arquivo app.py. Faça o mesmo para o campo "app.config['RECAPTCHA_PRIVATE_KEY']".
4 - Dentro da classe LoginForm, crie os campos do formulário de acordo com os exemplos que já estão no arquivo('Variavel' = 'Tipo do campo'('Nome do campo')).Talvez seja necessário importar mais tipos de campo, de acordo com a necessidade, como está na linha 3 do arquivo app.py. Não apagar 'recaptcha = RecaptchaField()'.
5 - Feito o passo 4, crie o formulário em seu arquivo HTML, como segue no exemplo do arquivo form.html. Para cada campo criado(Passo 4), invoque no seu arquivo HTML usando '{{form.NomeDoCampo}}'. Não apague '{form.csrf_token}}' nem '{form.recaptcha}}', pois são necessários para renderizar a API.
